# covid19_emergency

緊急事態宣言・まん延防止等重点措置の期間及び区域の情報を返す API です。

政府の運営する[新型コロナウイルス感染症対策](https://corona.go.jp/emergency/)の HP から GitHub Actions で定期的にスクレイピングし json ファイルを生成させています。
